Gerar Título / Boleto


# Exemplo curl
curl \
-X POST http://solicitarurl.com/rest/MrTitPR \
--header 'Content-Type: application/json' \
--header 'Authorization: Basic xxx' \
--data '[   
    {   
        "filial":"0101",
        "codigo":"000001",
        "loja":"01",
        "valor":100,
        "vencimento":"20/11/2023",
        "natureza":"000000000",
        "banco":"000",
        "agencia":"0000",
        "conta":"000000",
        "processo":"FXDACP0000000841"
    }
]'
                

Exemplo retorno:

Com sucesso:

[
    {
        "status": true,
        "mensagem": "gerado com sucesso.",
        "filial": "0101",
        "codigo": "000001",
        "loja": "01",
        "valor": 100,
        "numero": "000000008",
        "prefixo": "LEC",
        "boleto": "XXXXXXXXXXXXXX"
    }
]


Sem sucesso:

[
    {
        "status": false,
        "mensagem": "campos obrigatorios - valor/",
        "filial": "",
        "codigo": "",
        "loja": "",
        "valor": 0,
        "numero": "",
        "prefixo": "",
        "boleto": ""
    }
]

BODY PARAMETERS

Campo Tipo Descrição
filial String (max 4) codigo da filial de geração do título
codigo String (max 6) codigo do cliente
loja String (max 2) loja do cliente
valor Decimal (max 16,2)
vencimento String (max 10)
natureza String (max 10)
banco String (max 3)
agencia String (max 5)
conta String (max 10)
processo String (max 25)
